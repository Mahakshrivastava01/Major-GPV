# Major-GPV
GPV stands for GitHub profile viewer, I created this website for ease of users who are beginners and specially for the users who doesn't have any idea about github.So that anyone can search for th existing users of github.It is a simple and interactive web application that allows users to search and view any GitHub profile with key details like repositories, followers, and activity.

KEY FEATURES:
🔎 Search any GitHub user by username
👤 View profile details (name, bio, avatar)
📊 Display followers, following, and repositories
📁 List of public repositories
⚡ Fast and responsive UI

TECH STACK USED HERE:
Frontend: React.js
API: GitHub REST API
Styling: CSS / Tailwind

API USED:
GITHUB API :https://api.github.com/users/${username}

How It Works:
User enters a GitHub username
App fetches data using GitHub API
Displays profile information and repositories
